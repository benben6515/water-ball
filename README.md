## p0 - 核心功能

### 會員系統

- **OAuth 註冊/登入**

  - Google OAuth 整合
  - Facebook OAuth 整合
  - 第三方帳號綁定機制

- **基本資料管理**

  - 暱稱、email
  - 等級系統（Lv.）
  - 性別、生日、地區
  - GitHub 連結
  - 職業
  - 突破道館記錄

- **第三方綁定**

  - Discord 帳號綁定
  - GitHub 帳號綁定

- **訂單紀錄**
  - 購買歷史查詢
  - 訂單狀態追蹤
  - 擁有的課程列表

### 課程影片系統

- **影片播放功能**

  - 線上串流播放
  - 播放控制（暫停、快轉、調整速度）
  - 影片品質選擇

- **自動播放機制**

  - 播完自動跳下一節
  - 倒數 5 秒顯示提示
  - 可取消自動跳轉

- **進度追蹤**
  - 自動儲存觀看進度
  - 影片播到 95% 時標記為完成
  - 從上次中斷處繼續播放

### 課程列表頁面

- **課程展示**

  - 課程卡片設計（封面圖片、標題、講師）
  - 課程擁有狀態顯示（"尚未擁有" / "已擁有"）
  - 課程分類與篩選

- **課程資訊**
  - 課程簡介
  - 影片數量與總時長
  - 講師資訊
  - 購買按鈕

## p1 - 其他功能

### 🎮 遊戲化系統

#### 排行榜系統

- **本週成長榜**

  - 用戶排名顯示
  - 即時更新機制
  - 顯示前 N 名用戶

- **等級系統**

  - Lv. 等級顯示
  - 經驗值計算與升級機制
  - 等級對應權限或獎勵

- **成長指標追蹤**
  - 學習時數統計
  - 完成課程數量
  - 任務完成進度

#### 學習路徑/任務系統

- **10 項關鍵任務**

  - 新手任務（入門引導）
  - 黑段任務（進階挑戰）
  - 任務描述與完成條件

- **漸進式解鎖機制**

  - 前置任務要求
  - 任務依賴關係
  - 解鎖提示與引導

- **道館挑戰系統**

  - 不同難度的道館
  - 挑戰條件與獎勵
  - 突破道館記錄

- **任務完成條件**
  - 觀看指定影片
  - Discord 社群分享
  - 完成實戰題目
  - 提交作業或專案

#### 課程副本系統

- **8 個主要副本（章節）**

  - 副本零至副本七的層級設計
  - 每個副本包含多個影片單元
  - 總計 49 部影片內容

- **實戰題目系統**

  - 程式練習題目
  - 設計模式應用題
  - 作業提交與評分

- **難度分級**
  - ★ 至 ★★★★ 難度標示
  - 由淺入深的學習路徑
  - 難度對應學習目標
